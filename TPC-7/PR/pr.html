Abstract

Na diretoria JSONs encontram-se os ficheiro em JSON que contêm informação recolhida da Dbpédia.

O ficheiro script.js é um ficheiro em JavaScript que transforma os ficheiros JSON no ficheiro em Turtle individuos.ttl da diretoria TurtleFiles. Para tal ser possível é necessário executar o comando node script.js > individuos.ttl.

Após a criação do ficheiros individuos.ttl os conteudos deste são juntados aos conteúdos do ficheiro cinema.ttl, (representativo da parte estrutural da ontologia) para originar o ficheiro final_file.ttl que representa a ontologia na sua totalidade.
