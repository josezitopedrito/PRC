Abstract

A informação no ficheiro arquivo-musica-digital.xml em formato XML foi convertida, utilizando um ficheiro XSLT musica.xsl, para o formato Turtle, individuos.ttl. Entretanto outro ficheiro em formato Turtle, estrutura.ttl, foi gerado com a parte estrutural da ontologia, com o auxílio do Protégé.

No final são juntados os dois ficheiros em formato Turtle num único ficheiro, final.ttl, representativo da ontologia de forma integral.

Após a criação deste ficheiro, este é importado para o GraphDB. Sobre a base de dados criada são aplicadas as queries do ficheiro queries.txt
