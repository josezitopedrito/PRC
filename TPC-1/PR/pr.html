Abstract

Primeiramente foi gerado um ficheiro CSV, PRI2019-Avaliacao.csv, a partir da informação apresentada pelo docente de PRI na plataforma e-learning. Alguma informação dos alunos foi recolhida desse ficheiro CSV, sendo depois transferida para um outro ficheiro em formato XML studentsPRI.xml, com ajuda de um script elaborado em Python, extracao.py.

A informação em formato XML foi convertida, utilizando um ficheiro XSLT students.xsl, para o formato Turtle, students.ttl. Entretanto outro ficheiro em formato Turtle, proto.ttl, foi gerado com a parte estrutural da ontologia, com o auxílio do Protégé.

No final são juntados os dois ficheiros em formato Turtle num único ficheiro, TPC1.ttl, representativo da ontologia de forma integral.
